📅 Calendar Reminder App

A desktop-based reminder management application built using Python.
This app helps users schedule, store, and manage reminders with an interactive calendar interface.

Features

✅ Interactive Calendar View
✅ Add New Reminders
✅ Delete Existing Reminders
✅ Store Data Permanently using SQLite
✅ Auto-load Saved Reminders on Startup
✅ User-friendly Desktop GUI


🛠️ Technologies Used

* Python
* Tkinter
* SQLite
* tkcalendar
